# trying-pyramid
Trying out Pyramid python framework.

# How to:

```
    # Create VirtualEnv
    python3 -m venv env

    env/bin/pip install
    env/bin/pip install -e .

    # Run the server
    env/bin/pserve development.ini --reload

    # Run unittest
    env/bin/py.test tutorial/tests.py -q

```
